# Expanded Style Prompt Atlas

This file expands the first 180 prompts with additional design styles and narrower lanes. Use the same placeholder convention:

```text
{{SUBJECT}}
```

Each prompt is designed for image generation. Replace `{{SUBJECT}}` with the scene, product, interface, poster, interior, character, or object you want.
