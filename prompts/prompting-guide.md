# Prompting Guide For Design Styles

## Prompt Formula

```text
Create {{SUBJECT}} in [STYLE NAME] style: [palette], [shape language], [material/texture], [composition], [lighting], [typography or interface pattern when relevant], [era/culture signal]. Avoid [wrong adjacent styles], watermarks, logos, and unreadable fake text.
```

## Why The Placeholder Matters

`{{SUBJECT}}` lets one style prompt work for many outputs:

- poster
- website hero
- mobile app screen
- packaging design
- interior
- character
- product render
- editorial illustration
- game environment
- fashion lookbook

## Prompt Strength Scale

- **Light style transfer:** "with subtle Art Deco influence"
- **Clear style:** "in Art Deco style"
- **Strong style:** "fully Art Deco: stepped geometry, sunburst motifs, lacquered black and gold, symmetrical luxury composition"
- **Hybrid style:** "Cyberpunk x Swiss International: neon-dark atmosphere but strict grid typography and restrained information hierarchy"

## Safe Style Naming

Use public movements, eras, visual systems, materials, and subcultures. Avoid asking for the exact style of a living artist or active studio. When a known reference is tempting, translate it into ingredients:

- Instead of a specific director: "symmetrical pastel cinema, deadpan framing, centered architecture, careful color blocking."
- Instead of a specific animation studio: "soft pastoral anime watercolor, gentle rounded forms, warm natural light, hand-painted backgrounds."
- Instead of a specific brand: "premium monochrome fashion editorial, sparse serif typography, high-contrast studio lighting."

## Common Negative Prompt

```text
Avoid watermarks, logos, brand names, unreadable fake text, accidental extra limbs, messy typography, over-crowded composition, and generic stock-photo lighting.
```
